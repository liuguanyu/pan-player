## 分析工具使用方法

1. 将截图保存到 `reference_materials/` 目录
2. 按照以下命名规范：
   - `main_scene.png` - 主画面
   - `jumping.png` - 跳跃状态
   - `shaking_head.png` - 摇头状态
   - `breathing.png` - 呼吸状态
   - `spotlight_bright.png` - 聚光灯亮
   - `spotlight_dim.png` - 聚光灯暗

## 代码优化建议

根据截图分析结果，我们可以调整以下参数：

1. **身体参数**：
   ```javascript
   // 身体椭圆尺寸
   bodyWidth: 90,
   bodyHeight: 70,
   ```

2. **头部参数**：
   ```javascript
   // 头部椭圆尺寸和位置
   headWidth: 50,
   headHeight: 70,
   headOffsetX: -35,
   headOffsetY: -20,
   ```

3. **动画参数**：
   ```javascript
   // 跳跃幅度
   jumpAmplitude: 40,
   // 摇头幅度
   shakeAmplitude: 30,
   // 呼吸缩放
   breatheScale: 0.2,
   ```
